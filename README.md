# Kinetic WOL

Proyecto Android nativo para despertar dispositivos Wake-on-LAN desde el teléfono y exponer esa acción al asistente mediante App Actions.

## Base técnica

- Kotlin
- Jetpack Compose
- Material 3
- Room
- Android 16 o superior
